# site-ong-gatos

Projeto de site para uma ONG de gatos, desenvolvido com **HTML, CSS, JavaScript, PHP e MySQL**. O sistema permite exibir informações sobre adoção, cadastro de gatos e administração de usuários.  

---

## Tecnologias Utilizadas

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Banco de Dados:** MySQL  
- **Servidor Local:** XAMPP
